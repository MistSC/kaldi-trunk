Detailed Description

template<class Arc>
class fst::UnweightedNgramFst< Arc >

The class UnweightedNgramFst is a DeterministicOnDemandFst whose states encode an n-gram history.

Conceptually, for n-gram order n and k labels, the FST is an unweighted acceptor with about k^(n-1) states (ignoring end effects). However, the FST is created on demand and doesn't need the label vocabulary; GetArc matches on any input label. This class is primarily used by ComposeDeterministicOnDemand to expand the n-gram history of lattices.
