std::vector<Access> accesses

Records the indexes of commands that access the matrix, and the type (read, read/write, write).

It will be sorted on command index with only one record per command. Note: a write to only a part of the matrix (i.e. a submatrix that isn't the whole thing) will be recorded as an access of type read/write. An allocation with zeroing is recorded as a write access.
