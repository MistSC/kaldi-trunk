Detailed Description

This class is responsible for parsing input xx=yyy a=b c empty= f-oo=bar ba_z=123 bing='a b c' baz="a b c d='a b' e" and giving you access to the fields, in this case.

xx->yyy, a->"b c", empty->"", f-oo->bar, ba_z->"123", bing->"a b c", baz->"a b c d='a b' e"

Key values may contain -_a-zA-Z0-9, but must begin with a-zA-Z_.
