44 /* ContextDependency is quite a generic decision tree.
45 
46  It does not actually do very much-- all the magic is in the EventMap object.
47  All this class does is to encode the phone context as a sequence of events, and
48  pass this to the EventMap object to turn into what it will interpret as a
49  vector of pdfs.
50 
51  Different versions of the ContextDependency class that are written in the future may
52  have slightly different interfaces and pass more stuff in as events, to the
53  EventMap object.
54 
55  In order to separate the process of training decision trees from the process
56  of actually using them, we do not put any training code into the ContextDependency class.
57  */
