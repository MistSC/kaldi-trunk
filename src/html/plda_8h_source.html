37 /* This code
38  implements Probabilistic Linear Discriminant Analysis: see
39  "Probabilistic Linear Discriminant Analysis" by Sergey Ioffe, ECCV 2006.
40  At least, that was the inspiration. The E-M is an efficient method
41  that I derived myself (note: it could be made even more efficient but
42  it doesn't seem to be necessary as it's already very fast).
43 
44  This implementation of PLDA only supports estimating with a between-class
45  dimension equal to the feature dimension. If you want a between-class
46  covariance that has a lower dimension, you can just remove the smallest
47  elements of the diagonalized between-class covariance matrix. This is not
48  100% exact (wouldn't give you as good likelihood as E-M estimation with that
49  dimension) but it's close enough. */
50
