395 // This code obtains periodically (for each "ivector_period" frames, e.g. 10
396 // frames), an estimate of the iVector including all frames up to that point.
397 // This emulates what you could do in an online/streaming algorithm; its use is
398 // for neural network training in a way that's matched to online decoding.
399 // [note: I don't believe we are currently using the program,
400 // ivector-extract-online.cc, that calls this function, in any of the scripts.].
401 // Caution: this program outputs the raw iVectors, where the first component
402 // will generally be very positive. You probably want to subtract PriorOffset()
403 // from the first element of each row of the output before writing it out.
404 // For num_cg_iters, we suggest 15. It can be a positive number (more -> more
405 // exact, less -> faster), or if it's negative it will do the optimization
406 // exactly each time which is slower.
407 // It returns the objective function improvement per frame from the "default" iVector to
408 // the last iVector estimated.

412  const IvectorExtractor &extractor,
413  int32 ivector_period,
