Detailed Description

This class basically does the same job as the function DecodeUtteranceLatticeFaster, but in a way that allows us to build a multi-threaded command line program more easily, using code in ../thread/kaldi-task-sequence.h.

The main computation takes place in operator (), and the output happens in the destructor.
